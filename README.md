#TeaPHP
 
*喝着茶，唱着歌，就能把活给干了*

*每一段代码，都能自动适应多个用途*
 
~~~
Your Code  |----- JSON Data
           |----- Remote Service
           |----- API Docs
           |----- HTML View  --|----- AngularJS
                               |----- Vue.js
                               | ......
          			
~~~
